## How It Works

### Language Switcher
- Located in top-right corner of all pages
- Two buttons: **EN** | **VI**
- Current language highlighted in red
- Preference saved to localStorage
- Persists across sessions

### Translation Mechanism
```javascript
// HTML with translation attribute
<h2 data-i18n="dashboard.price_chart">Price Chart</h2>

// When user clicks VI button:
// JavaScript finds all [data-i18n] elements
// Looks up key in translation dictionary
// Replaces text content with Vietnamese translation

// Result:
```
